# Genomic Imprinting Model

Simulación del modelo matemático de expresión génica bajo estrés salino.

## Modelo

dG/dt = rG(1 - G/K) - sG

## Características

- Simulación en tiempo real
- Interfaz científica oscura
- Ajuste de parámetros dinámicos
- Visualización gráfica

## Uso

1. Abrir index.html
2. Ajustar parámetros
3. Ejecutar simulación
